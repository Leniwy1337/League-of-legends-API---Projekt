### Opis Projektu

Aplikacja laczy sie z Data Dragon API i pobiera dane o bohaterach.
W projekcie używamy lokalnej bazy danych "hive_ce" do przechowywania danych.
Na ekranie głównym znajduje sie prezentacja ogolnych informacji o postaciach.
Na drugim ekranie wyświetlają sie szczegółwoe informacje o postaci.
